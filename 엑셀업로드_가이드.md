# 엑셀 파일 업로드 가이드

## 📋 엑셀 파일 준비

### 1. 엑셀 파일 생성
- Microsoft Excel 또는 Google Sheets 사용
- 파일 형식: `.xlsx` 또는 `.xls`

### 2. 시트 구성

#### 필수 시트: 가맹점
**시트명**: "가맹점" 또는 "franchise"

| 컬럼명 (한글) | 컬럼명 (영문) | 필수 | 설명 | 예시 |
|--------------|--------------|------|------|------|
| 지역 | region | ✅ | yongin 또는 gangdong | yongin |
| 상호 | name | ✅ | 가맹점 이름 | 미래엔수학 용인점 |
| 주소 | address | ✅ | 전체 주소 | 경기도 용인시 처인구... |
| 원장명 | director | ✅ | 원장 이름 | 홍길동 |
| 회원수 | members | ✅ | 숫자만 | 150 |
| 전화번호 | phone | ❌ | 010-1234-5678 형식 | 010-1234-5678 |
| 우편번호 | zipCode | ❌ | 5자리 숫자 | 12345 |
| 계약일 | contractDate | ❌ | YYYY-MM-DD 또는 YYYYMMDD | 2024-01-15 |
| 네이버플레이스ID | naverPlaceId | ❌ | 네이버 플레이스 ID | 12345678 |

#### 선택 시트: 학원상권
**시트명**: "학원상권" 또는 "academy"

| 컬럼명 (한글) | 컬럼명 (영문) | 필수 | 설명 | 예시 |
|--------------|--------------|------|------|------|
| 지역 | region | ✅ | yongin 또는 gangdong | yongin |
| 학원상권명 | name | ✅ | 학원상권 이름 | 용인 학원가 |
| 주소 | address | ✅ | 전체 주소 | 경기도 용인시... |
| 반경(미터) | radius | ❌ | 미터 단위 숫자 (기본값: 500) | 500 |

## 📤 업로드 방법

### 방법 1: 드래그 앤 드롭
1. 엑셀 파일을 준비
2. 브라우저에서 `excel-upload.html` 페이지 열기
3. 엑셀 파일을 업로드 영역에 드래그 앤 드롭

### 방법 2: 파일 선택
1. 브라우저에서 `excel-upload.html` 페이지 열기
2. "파일 선택" 버튼 클릭
3. 엑셀 파일 선택

## 📥 다운로드 및 적용

1. 업로드 후 미리보기 확인
2. "data.js 다운로드" 버튼 클릭
3. 다운로드한 `data.js` 파일을 프로젝트 폴더에 복사
4. 기존 `data.js` 파일을 덮어쓰기
5. `index.html` 새로고침

## ⚠️ 주의사항

- **지역 필드**: 반드시 소문자로 "yongin" 또는 "gangdong" 입력
- **회원수**: 숫자만 입력 (문자 입력 시 0으로 처리됨)
- **계약일**: YYYY-MM-DD 형식 권장 (예: 2024-01-15)
- **시트명**: 정확히 일치해야 함 (대소문자 구분 없음)
- **컬럼명**: 한글 또는 영문 모두 가능
- **초등학교**: 별도 입력 불필요 (네이버 지도에서 자동 검색)

## 💡 팁

- 엑셀에서 데이터를 입력할 때 첫 번째 행은 컬럼명으로 사용됩니다
- 빈 셀은 선택사항 필드이므로 비워도 됩니다
- 네이버 플레이스 ID가 없어도 자동으로 검색 링크가 생성됩니다
