## 5. 문제 해결

### 지도가 안 보이는 경우
1. **브라우저 콘솔 확인 (F12)**
   - 에러 메시지 확인
   - "API 인증이 실패했습니다" 메시지가 있는지 확인

2. **Web 서비스 URL 확인**
   - Vercel URL이 정확히 등록되었는지 확인
   - 마지막 슬래시(/) 없이 등록했는지 확인
   - 예: `https://miraen-map.vercel.app` ✅
   - 예: `https://miraen-map.vercel.app/` ❌

3. **저장 후 대기 시간**
   - Application 저장 후 최소 10분, 최대 30분 대기
   - 저장 후 즉시 반영되지 않을 수 있음

4. **브라우저 캐시 삭제**
   - Ctrl + Shift + Delete
   - 캐시된 이미지 및 파일 삭제
   - 시크릿 모드에서 테스트

### 여전히 안 되는 경우
- 네이버 클라우드 플랫폼 고객센터 문의
- 신규 Maps API로 전환 고려
