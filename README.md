# 미래엔수학 가맹점 지도

용인지사 및 강동송파하남지사 가맹점 위치를 지도에 표시하고, 초등학교 주변 및 학원상권 분석을 위한 웹 애플리케이션입니다.

## 기능

- 🗺️ 네이버 지도 기반 가맹점 위치 표시
- 🏫 초등학교 위치 표시
- 📍 주요 학원상권 영역 표시
- 🔍 회원수 필터링 기능
- 📊 가맹점 정보 상세 보기 (상호, 주소, 원장명, 회원수)
- 🎯 지역별 필터링 (용인지사 / 강동송파하남지사)

## 사용 방법

### 1. 네이버 지도 API 설정

1. [네이버 클라우드 플랫폼](https://www.ncloud.com/)에 가입
2. Console > AI·NAVER API > AI·NAVER API > Application 등록
3. Maps API 선택 후 Application 등록
4. Client ID 발급
5. `index.html` 파일의 7번째 줄에서 `YOUR_CLIENT_ID`를 발급받은 Client ID로 변경

```html
<script type="text/javascript" src="https://oapi.map.naver.com/openapi/v3/maps.js?ncpClientId=YOUR_CLIENT_ID"></script>
```

### 2. 데이터 입력

`data.js` 파일에 가맹점, 초등학교, 학원상권 데이터를 입력하세요.

#### 가맹점 데이터 형식
```javascript
{
    name: "가맹점명",
    address: "주소",
    director: "원장명",
    members: 100,  // 회원수
    lat: 37.2411,  // 위도 (선택사항, 주소만 있어도 자동 변환)
    lng: 127.1776  // 경도 (선택사항, 주소만 있어도 자동 변환)
}
```

#### 초등학교 데이터 형식
```javascript
{
    name: "초등학교명",
    address: "주소",
    lat: 37.2411,  // 위도 (선택사항)
    lng: 127.1776  // 경도 (선택사항)
}
```

#### 학원상권 데이터 형식
```javascript
{
    name: "학원상권명",
    address: "주소",
    lat: 37.2411,  // 위도 (선택사항)
    lng: 127.1776, // 경도 (선택사항)
    radius: 500    // 반경 (미터)
}
```

### 3. 실행

웹 브라우저에서 `index.html` 파일을 열면 됩니다.

로컬 서버를 사용하려면:

```bash
# Python 3
python -m http.server 8000

# 또는 Node.js (http-server 설치 필요)
npx http-server
```

그 후 브라우저에서 `http://localhost:8000` 접속

## 주요 기능 설명

### 필터 기능
- **가맹점 표시**: 가맹점 마커 표시/숨김
- **초등학교 표시**: 초등학교 마커 표시/숨김
- **학원상권 표시**: 학원상권 영역 표시/숨김
- **회원수 필터**: 최소 회원수 이상인 가맹점만 표시

### 마커 설명
- 🔴 **빨간색 원**: 가맹점
- 🔵 **청록색 원**: 초등학교
- 🟡 **노란색 다이아몬드**: 학원상권 중심점
- 🟡 **노란색 원형 영역**: 학원상권 반경

### 사용 팁
- 좌측 사이드바에서 가맹점을 클릭하면 해당 위치로 지도가 이동합니다
- 지도上的 마커를 클릭하면 상세 정보를 볼 수 있습니다
- 하단 정보 창에서 가맹점의 상세 정보를 확인할 수 있습니다

## 주의사항

- 네이버 지도 API는 일일 사용량 제한이 있습니다
- 주소만 입력해도 자동으로 좌표로 변환되지만, 정확한 좌표를 직접 입력하는 것이 더 정확합니다
- 학원상권 반경은 미터 단위로 설정합니다
